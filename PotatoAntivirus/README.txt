# 🥔 Potato-Antivirus v1.0
**The ultimate "potato" protection for your PC.**

Potato-Antivirus — это легковесный комплекс мониторинга и защиты от локальных DoS-атак (форк-бомб). Создан для тех, чьи компьютеры не тянут тяжелые антивирусы, но требуют контроля над системными ресурсами.

## ✨ Основные возможности
- 🖥️ **Cyber-Shield Dashboard**: Консольный мониторинг ресурсов в реальном времени (Batch).
- 🛡️ **Guard HUD**: Интерактивный оверлей с кнопками управления и авто-защитой (AutoHotkey).
- ☢️ **Force Stop**: Мгновенное уничтожение опасных процессов одной кнопкой или через Alt+F12.
- 📁 **Lab Explorer**: Быстрый доступ к вашим "исследовательским" папкам (v1, v2, archive).
- 🚀 **Anti-Fork Logic**: Автоматическая блокировка размножения CMD-процессов (лимит: 15).

## 🛠️ Инструкция по установке
1. Установите [AutoHotkey v1.1](https://www.autohotkey.com).
2. Скачайте файлы из этого репозитория.
3. Запустите `Guard.ahk` для активации пульта управления.
4. Нажмите `> OPEN DASHBOARD`, чтобы запустить монитор ресурсов.
